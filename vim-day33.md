# vim-day33 
## 今日目标
> 学习chrome页面内的搜索与查找

## 操作
+ `o`，显示多功能搜索框
  + 直接输入link
  + 历史记录
  + 搜索
  + 搜索关键字
    + `g`
    + `gh`
    + `b`
    + `bz`
    + 
+ `O`，同`o`，只不过点击之后会新开页面
+ `b`，显示搜索框并搜索收藏夹的内容
+ `B`，同`b`，只不过点击之后会新开页面
+ `ge`，显示搜索框/编辑当前网址
+ `gE`，同`ge`，只不过点击之后会新开页面
+ `T`，显示当前浏览器的全部打开的标签页
+ `/`，当前页面全局搜索
  + `n`，往下翻记录
  + `N`，网上翻记录

## 扩展
+ 修改默认搜索引擎
+ 在chrome:///页面上运行
  + 在chrome:///flags页面，修改extensions-on-chrome-urls为enabled
