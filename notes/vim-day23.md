# vim-day23

## 今日目标
> 发现使用快捷键的场景

## 操作
+ 资源管理器
  + `mac`环境下，`command + b`，显示/隐藏资源管理器
  + `win`环境下，`Alt + b`，显示资源管理器
+ 使用`live sever`起服务
  + `mac`环境下，`command + l` + `command + o` 启动服务；`command + l` + `command + c` 关闭服务
  + `win`环境下，`Alt + l` + `Alt + o` 启动服务；`Alt + l` + `Alt + c` 关闭服务
+ 预览 `md`
  + `mac`环境下
    + `command + k + v`，打开侧边预览
    + `command + shift + v`，打开预览
  + `win`环境下
    + `alt + m`，打开侧边预览(通过改键配置)
    + `Ctrl + shift + v`，打开预览
+ 显示扩展
  + `mac`环境下，`command + shift + x`
  + `win`环境下，`Alt + shift + x`
+ 显示真实文件详情
  + `mac`环境下，`command + k + r`
+ 复制活跃文件路径
  + `mac`环境下，`command + k + p`
  + `win`环境下，`Alt + k + p`
