# vim-day07
## 今日目标
> 在单文件中更快地移动 

## 操作
+ 滚动
  + 按 `Ctrl + f` 键（forward），向下滚动一屏
  + 按 `Ctrl + b` 键（backward），向上滚动一屏
  + 按 `Ctrl + d` 键，向下滚动半屏
  + 按 `Ctrl + u` 键，向上滚动半屏
  + 按 `Ctrl + e` 键，向下滚动一行
  + 按 `Ctrl + y` 键，向上滚动一行
  + 基于配置（更容易记忆）
    + 按 `shift + k` 键，向上滚动 5 行
    + 按 `shift + j` 键，向下滚动 5 行
    + 按 `shift + k` 键，向上滚动 5 行
+ `zz`，将当前行置于屏幕中央
+ `zt`，将当前行置于屏幕顶部附近
+ `zb`，将当前行置于屏幕底部附近
+ `gg`，跳转到文件首
+ 按 `shift + g` 键，跳转到文件尾
+ 跳转到指定行
  + 行数 + `gg`
  + 行数 + `shift + g`
