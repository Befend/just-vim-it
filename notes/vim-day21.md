# vim-day21

## 今日目标
> 掌握搜索

## 操作
+ 全局搜索
  + `Ctrl + shift + f`，打开 `vscode` 的全局搜索栏
  + `Ctrl + up/down`，选中搜索栏的输入框
  + `Ctrl + shift + j`，查看搜索内容
+ 查找单一工作空间，
  + `Ctrl + t`
+ 查找单一文件
  + `Ctrl + shift + o` / `Ctrl + p + @`，搜索框出现`@`，搜索变量
  + `Ctrl + p + :`，按照类别分组
+ `Ctrl + shift + p`， 查找命令
+ `Ctrl + p`，快速定位文件
+ 切换文件，
  + `vim环境` 下，两个文件的快速跳转，`Ctrl + i/o`
