# vim-day37

## 今日目标
> 学习chrome的debugger

## 操作
+ 打断点
  + 在 `mac` 系统下，`command + b` 或者 `command + shift + b`
  + 在 `Windows` 系统下，`Ctrl + shift + b`
+ `f10`，Step over
  + 在 `mac` 系统下，`command + '`
+ `f11`，Step into
  + 在 `mac` 系统下，`command + ;`
+ `f5`，停止当前的断点
  + 在 `mac` 系统下，`command + \`
+ Deactivate breakpoint
  + 在 `mac` 系统下，`command + f8`
+ 把选中的代码打印出来
  + 在 `mac` 系统下，`shift + command + e`
+ 跳转到函数
  + 在 `mac` 系统下，`shift + command + o`
+ `Ctrl + g`，跳转到指定行m
