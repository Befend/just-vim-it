# vim-day30
## 今日目标
> 学习vscode的debugger

## 操作
+ `fn + 5`，debugger
+ `shift + cmd + 0`，重启debugger
+ 打断点
  + mac系统下，`shift + cmd + 9`
  + windows系统下，`fn + 9`
  + `editor.debug.action.toggleInlineBreakpoint`
+ step over
  + mac系统下，`cmd + '`
  + windows系统下，`shift + '`
  + `workbench.action.debug.stepOver`
+ step into
  + mac系统下，`shift + cmd + '`
  + windows系统下，`shift + ctrl + '`
  + `workbench.action.debug.stepInto`
+ step out
  + `f5`
