# vim-day04
## 今日目标
> 更有效率的处理单字符 & undo/redo

## 操作
+ 按 `x` 键，删除光标所在的字符
+ 按 `shift + x` 键，删除光标前的字符
+ 按 `s` 键，删除当前光标的字符并进入 `insert模式`
+ 按 `shift + s` 键，删除当前光标所行并进入 `insert模式`
+ 按 `r` 键，替换一个字符
+ 按 `shift + r` 键，替换多个字符
+ 按 `u` 键，undo（向前撤销）
+ 按 `Ctrl + r` 键，redo（向后撤销）
