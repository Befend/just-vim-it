# vim-day01
## 今日目标
> 熟悉vim的基本操作
## 安装
>  在vscode中安装vim插件

## normal模式与insert模式
+ 按 `i` 键，从`insert模式` 切换到 `normal模式`，
+ 按 `Ctrl + [` 或者 `esc` 键，从`normal模式` 切换到 `insert模式`

## 基本操作
### 光标移动（在 `normal模式` 下）
+ 按 `h` 键，向 `左` 移动
+ 按 `l` 键，向 `右` 移动
+ 按 `k` 键，向 `上` 移动
+ 按 `l` 键，向 `下` 移动
+ 按 `i` 键，光标前插入，
+ 按 `a` 键，光标后插入

## 扩展
+ 在终端如何使用vim / 如何退出
```powershell
:wq # 保存修改并退出
:q! # 强制退出
```
+ Ctrl和caps位置调换
